# portfolio# 
**Cyber-Infused Personal Portfolio**

A vibrant, highly animated, and fully responsive developer portfolio website designed with a **Neon Pink** and **Cyber-Light Blue** color scheme. Built using HTML5, CSS3, Bootstrap 5, and JavaScript to showcase skills, engineering projects, and leadership milestones.

The data presented in this portfolio is generated directly from `Katta Hepsi(resume).pdf`[cite: 1].

---

##  Features

- **Cyber Dynamic Palette:** Bold dark theme accented with vivid neon pink (`#ff007f`) and electric cyber-blue (`#00f0ff`) gradients.
- **Scroll Reveal Animations:** Sections smoothly fade and transition into view as the user scrolls down the page.
- **Interactive Micro-Animations:** Project cards expand dynamically, buttons glow on hover, and tech icons gently float in place.
- **Responsive Layout:** Engineered with Bootstrap 5 to look completely clean and tailored on mobile, tablet, and desktop viewports.
- **Quick Profiles Integration:** Pre-configured anchor tags to link your active GitHub, LinkedIn, LeetCode, and GeeksforGeeks profiles directly.

---

##  Built With

* **Markup & Core Styling:** HTML5 & CSS3
* **Layout & Grid Framework:** Bootstrap v5.3
* **Iconography:** Font Awesome v6.4
* **Motion & Effects:** Animate.css & JavaScript Intersection Observer API

## Getting Started

Follow these simple steps to run the website locally or update it with your links:

### 1. Clone or Download the Project
Download the repository files to your local machine, ensuring you keep your structure intact.

### 2. Update Your Profile Links
